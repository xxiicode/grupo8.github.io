# Workshop Node.js #23572 Grupo 8

## Instalacion

1 - Descargar e instalar node.js
verificar y chekear version con:

```
node -v
```

2- Instalar en version local todas las dependencaias y crea los node_modules

```
npm install
```

## Ejecutar

modo produccion:
```
node index.js
```

modo Dev (+nodemon):
```
npm run dev
```
